# portfolio / about me

small little static site that works as a online CV / about me page

## setting up & using

developed with node v20.12.2 & npm v10.5.0

install depedencies

`npm i`

run locally with hot reload etc.

`npm start`

build for prod & serve

`npm run generate`

`npm run serve`

cleanup nuxt generated files & caches

`npm run cleanup`
